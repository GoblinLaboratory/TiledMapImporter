Remarks
This method can be a common cause of NullReferenceExceptions. If the Tileset textures are not properly loaded, the Map object cannot be drawn. If the files cannot be located in the project, there will be a Null Reference.
Examples
If your project stores all Tileset textures inside a "tilesets" folder in your Content directory, you could alter the filepath to automatically look in that folder.
