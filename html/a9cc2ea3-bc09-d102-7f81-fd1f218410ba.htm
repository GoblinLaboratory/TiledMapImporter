The TileLayer type exposes the following members.

Properties
  NameDescription
Data
Contains the global ids for the layer.
Height
The height of the layer in tiles. Traditionally required, but as of Tiled Qt always the same as the map width.
Name
The name of the layer.
Opacity
The opacity of the layer as a value from 0 to 1. Defaults to 1.
Props
Custom properties for the layer object.
Visible
Whether the layer is shown (1) or hidden (0). Defaults to 1.
Width
The width of the layer in tiles. Traditionally required, but as of Tiled Qt always the same as the map width.
X
The x coordinate of the layer in tiles. Defaults to 0 and can no longer be changed in Tiled Qt.
Y
The y coordinate of the layer in tiles. Defaults to 0 and can no longer be changed in Tiled Qt.
